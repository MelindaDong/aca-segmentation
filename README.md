# ACA-Segmentation

Official implementation of **ACA: Post-hoc Adaptive Logit Alignment for 3D Medical Image Segmentation**.

Accepted by MICCAI 2026.

---

## 🔍 Overview

This repository contains the official PyTorch implementation of ACA for 3D medical image segmentation.
The method focuses on post-hoc logit alignment to improve model generalization and robustness in 3D medical imaging segmentation tasks.
